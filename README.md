## Sobre este repositório

Este repositório é o meu diário público de aprendizado em C#. Cada mini projeto representa um conceito estudado, erros, um bug identificado e resolvido.

O foco é aprender C# com aplicações voltadas para **desenvolvimento de jogos**

## Como rodar

```bash
# Entre na pasta do projeto desejado
cd "mini projects/NomeDoProjeto"

# Copie os comandos e cole no VS Code
ou qualquer outro editor de código de sua preferência

# Rode!
dotnet run
```
